# Telecommunication Network Design (GNS3)

Project Overview
This project demonstrates a secure enterprise telecommunication network designed and simulated using GNS3. 
The network uses VLANs to isolate traffic between departments and DHCP services for automatic IP address allocation. 
Security best practices such as encrypted passwords on routers and switches are also implemented.


Tools & Technologies
- GNS3 (Network Simulator)
- Routers and Switches
- VLAN (Virtual Local Area Network)
- DHCP (Dynamic Host Configuration Protocol)
- Device Security (Encrypted Passwords)


Network Design
- Multiple VLANs are configured to represent different departments.
- VLANs provide traffic isolation and secure communication.
- A DHCP service assigns IP addresses dynamically to hosts.
- Routers and switches are secured using encrypted passwords.


Security Features
- VLAN-based traffic isolation
- Encrypted passwords on all networking devices
- Secure enterprise-style configuration


How to Run / Use This Project
To make this project work, follow the steps below:

1. Download or clone this repository to your local system.
2. Install GNS3 on your computer (GNS3 must already be installed).
3. Open GNS3 and import/open the project file.
4. Start all devices in the topology.
5. Verify:
   - VLAN communication
   - DHCP IP address assignment
   - Secure access to routers and switches

## Note: 
This project will only work inside GNS3. Without GNS3, the project cannot be played or tested.
